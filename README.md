# Learning Emscripten and asm.js

## How to Run
1. Command:

    ```
    emcc hello.c -o hello.html --preload-file ./asset -s NO_EXIT_RUNTIME=1
    ```
2. Open HTML pages in Firefox. If use Chrome, run ```emrun xxx.html``` first.
